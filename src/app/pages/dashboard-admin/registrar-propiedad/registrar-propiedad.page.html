Como registrar una propiedad

Para registrar una propiedad tienes que completar todos los apartados solicitados que se muestran en esta pantalla.

Cuando abres un apartado y lo cierras por error te aparecerá la opción de volver a reiniciar el apartado puedes reiniciarlos las veces que consideres necesarias.

Para publicar una propiedad en automático la misma aplicación te avisara que estas listo para continuar a la publicación.
